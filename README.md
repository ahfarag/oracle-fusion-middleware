# Oracle Fusion Middleware with Docker

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](./LICENSE)

A growing collection of practical, hands-on guides for running Oracle Fusion Middleware products locally using Docker — no complex installers, no environment headaches.

Each guide is self-contained and includes all the steps needed to get up and running from scratch.

## Prerequisites

All guides in this repo share these common requirements:

- [Docker](https://docs.docker.com/get-docker/) installed and running
- A free [Oracle account](https://profile.oracle.com/myprofile/account/create-account.jspx) to accept image licenses on the Oracle Container Registry
